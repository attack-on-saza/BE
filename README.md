## 📝 소개
이 앱은 전국 각 지역의 전설과 괴담을 수집·크롤링하고, AI를 통해 이야기를 재창조하여 사용자에게 제공합니다. 각 이야기의 주요 특징과 태그를 AI가 추출해 손쉽게 검색하고 공유할 수 있는 플랫폼입니다.

- 프로젝트 소개
- 프로젝트 화면 구성 또는 프로토 타입
- 프로젝트 API 설계
- 사용한 기술 스택
- 프로젝트 아키텍쳐
- 기술적 이슈와 해결 과정
- 프로젝트 팀원
